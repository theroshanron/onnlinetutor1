Complete Course Curriculum (12 Weeks • 36 Classes)

Schedule: Mondays, Tuesdays, Fridays • 30 minutes per session. Each class uses: 3‑min warm‑up → 10‑min demo → 12‑min guided practice → 5‑min recap & homework.

Month 1 Summary (Weeks 1–4)

  • Comfort with mouse & keyboard; basic shortcuts
  • Windows basics: Start menu, taskbar, settings
  • File & folder skills: create, name, move, copy, delete, restore
  • Use File Explorer & USB safely
Week 1 — Getting Started: What is a Computer? Desktop Tour & Input Basics

Weekly Focus: Concept of hardware vs software, turning a laptop on/off safely, desktop basics, mouse & keyboard introduction.

Day Class Objective Hands-on Homework (5–10 min)
Mon Understand computer parts & OS; learn power on/off; tour the desktop (icons, taskbar, Start menu). Identify ports, power button; open Start menu; pin an app to taskbar. Shut down & restart once; find and open Calculator from Start.
Tue Mouse basics: single/double/right-click; drag & drop. Keyboard layout awareness. Practice clicks on desktop icons; use touchpad gestures; type your name in Notepad. 5 minutes of typing practice.
Fri Window management: open/close/minimize; switch apps; volume & Wi‑Fi basics. Open two apps and Alt+Tab between them; connect/disconnect Wi‑Fi demo. Change desktop wallpaper.
Week 2 — Typing & Navigation: Text Selection, Copy/Paste, Essential Shortcuts
Day Class Objective Hands-on Homework
Mon Typing posture; home row; numbers & symbols. Type a short paragraph; practice Backspace/Delete. Type 5 lines about your day; save as typing_practice.txt.
Tue Select, copy, cut, paste; undo/redo. Copy a sentence between Notepad files; Ctrl+Z/Ctrl+Y practice. Re-type and format a given sentence.
Fri Shortcuts & screenshots. Take a screenshot and save to Pictures. Memorize 5 shortcuts.
Week 3 — Windows Basics: Settings, Apps (Install/Uninstall), Updates
Day Class Objective Hands-on Homework
Mon Settings tour: Wi‑Fi, Bluetooth, time/date, display, sound. Change sound & time format; connect Wi‑Fi. Switch light/dark mode.
Tue Install via Microsoft Store; pin/unpin; uninstall safely. Install Sticky Notes and uninstall it. Create a Start group with favorite apps.
Fri Windows Update; battery & power settings. Check updates; set screen timeout; learn Sleep vs Hibernate. Apply a custom power plan.
Week 4 — Files & Folders: Explorer, Organization, USB & Downloads
Day Class Objective Hands-on Homework
Mon Explorer tour: address bar, nav pane, views. Create Documents/Letters, Pictures/Family, Bills. Change view & sort by date.
Tue Create/rename/move/copy/delete; Recycle Bin & restore. Copy photos; restore a deleted test file. Rename 5 files with clear names.
Fri Downloads tips; USB save/eject; file types. Download a PDF; move to Documents; safely eject USB. Screenshot your final folder structure.

Month 2 Summary (Weeks 5–8)

  • Browser basics, search, tabs, bookmarks
  • Gmail account & etiquette, attachments, labels/filters
  • Google Drive for storage & sharing
  • Docs & Sheets fundamentals
Week 5 — Internet Basics: Chrome, Search, Bookmarks, Safer Browsing
Day Class Objective Hands-on Homework
Mon Chrome tour; tabs; restore closed tab. Open 3 sites; Ctrl+Shift+T to restore. Bookmark 3 favourite sites.
Tue Effective searching & results scanning. Search for "best free typing practice"; open results in new tabs. Download a sample file and find it in Downloads.
Fri HTTPS/padlock; pop‑ups; privacy basics. Check a URL for spelling; clear cache. Read one page on phishing basics.
Week 6 — Email & Communication: Gmail Account, Attachments, Etiquette
Day Class Objective Hands-on Homework
Mon Create Google account; Gmail tour. Send a test email to tutor; star it. Reply to tutor politely.
Tue Compose with subject/formatting; CC/BCC; attachments. Attach a PDF from Documents and send. Create label "Course" and move mails.
Fri Search; filters; signature; undo send. Make a filter to auto-label tutor emails. Write a reschedule request.
Week 7 — Google Drive & Docs: Cloud Storage, Formatting, Sharing
Day Class Objective Hands-on Homework
Mon Drive tour; upload files/folders; structure. Make Course>Docs and Course>Sheets folders. Star the main course folder.
Tue Docs formatting: headings, bold/italic, bullets. Write and format a one-page letter. Export to PDF and save to Drive + local.
Fri Share (Viewer/Commenter/Editor); comments. Share your letter with tutor; add a comment. Rename doc with date suffix.
Week 8 — Google Sheets: Data Entry, Simple Formulas, Sort/Filter
Day Class Objective Hands-on Homework
Mon Create a Sheet; format numbers & currency. Make a small expense table. Auto-fill dates; adjust column widths.
Tue Formulas: SUM, AVERAGE, MIN, MAX. Compute totals & averages. Add rows and recompute.
Fri Sort, Filter, Freeze top row; basic chart. Sort by Amount; filter; insert a bar chart. Share as Viewer.

Month 3 Summary (Weeks 9–12)

  • Create simple presentations (Slides) and present confidently
  • Join/schedule Google Meet; use mic/cam/chat; share screen
  • Recognize phishing & scams; use strong passwords and 2‑Step Verification
  • Capstone: Create a Doc + Sheet + Slides and email them; present on Meet
Week 9 — Google Slides: Themes, Layouts, Images & Presenting
Day Class Objective Hands-on Homework
Mon Create Slides; choose theme; title/content slides. Make 3 slides about a hobby. Reorder slides; add speaker notes.
Tue Insert images/shapes; align & distribute; transitions. Add 2 images; align neatly. Practice your talk.
Fri Present mode; export PDF; share link. Present to tutor for 2 minutes. Save PDF to Drive.
Week 10 — Google Meet: Join, Schedule, Share Screen
Day Class Objective Hands-on Homework
Mon Join via link; mic/cam; background; chat. Join a demo Meet; toggle mic/cam; send a chat. Test mic/cam before next class.
Tue Schedule from Calendar; invite someone. Create a 15‑min event and add tutor. Add a reminder notification.
Fri Share screen (entire screen/window/tab). Share Week 9 Slides for 2 minutes. Note 3 Meet etiquette rules.
Week 11 — Online Safety: Passwords, 2FA, Phishing, Backups
Day Class Objective Hands-on Homework
Mon Strong passwords; password managers (concept); enable Google 2FA. Turn on 2‑Step Verification. Update recovery phone/email.
Tue Spot phishing emails & fake websites. Identify red flags from samples. Write 3 signs of suspicious email.
Fri Privacy check-up; Windows Defender; backups. Run a quick scan; review privacy settings. Back up a key folder to Drive.
Week 12 — Capstone & Review: Integrate Skills, Assessment, Next Steps
Day Class Objective Hands-on Homework
Mon Capstone kickoff & checklist. Create: 1 Doc (letter), 1 Sheet (budget), 1 Slides (3–5 slides). Finish draft of all 3 files.
Tue Email & sharing practice. Email the 3 files (PDF/links) to tutor; set permissions. Rehearse a 2–3 minute talk.
Fri Final presentation & review. Present on Meet with screen share; feedback. Plan next learning goals.

30‑Minute Demo Class (Confidence‑Boosting Intro)

  1. Icebreaker (0–3 min): What do you want to do with a computer? (email, bills, photos)
  2. What is a computer? (3–7 min): Hardware vs software; keyboard/touchpad; ports.
  3. Mouse & typing basics (7–12 min): Click/double/right‑click; type your name in Notepad.
  4. Hands‑on task (12–20 min): File Explorer → create folder Practice → new text file → rename → delete & restore.
  5. Internet tour (20–26 min): Chrome → URL vs search → search "Taj Mahal" → open result → bookmark.
  6. Wrap‑up (26–30 min): Homework: create My First Folder, add hello.txt, bookmark google.com.
